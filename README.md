My Python Project

Короткий навчальний проєкт, створений для практики роботи з Git та Python.
Він демонструє базову взаємодію з користувачем, роботу з файлами та просту логіку.
Функціональність
-Виведення привітання користувачу
-Читання імен із файлу
-Відображення поточного часу
-Обробка кількох імен
-Структурований код на Python
Встановлення
1. Клонуйте репозиторій:
git clone https://github.com/USERNAME/REPOSITORY.git
2. Перейдіть у папку проєкту:
cd my-python-project
3. Запустіть скрипт:
python greetings.py
